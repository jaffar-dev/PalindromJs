# PalindromeJs
A simple JavaScript project to check whether a word or number is a palindrome.

## Live Link:
https://palindromecheckerjs.netlify.app/

## Features
- Checks strings
- Ignores case sensitivity
- Simple and clean logic

## How to Use
1. Clone the repo
2. Open index.html
3. Enter text to test
